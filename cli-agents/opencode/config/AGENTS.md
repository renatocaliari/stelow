# OpenCode Agent Configuration

This directory contains OpenCode-specific configuration and instructions.

## Commands

All cali-product-workflow commands are documented in `../../COMMANDS.md` (single source of truth).

### Available Commands

| Command | Description |
|---------|-------------|
| `/pw-start` | Start a new workflow |
| `/pw-menu` | Show workflow menu |
| `/pw-status` | Show current status |
| `/pw-help` | Get help |

## Plugin Installation

See `../README.md` for installation instructions.

## Configuration

The plugin requires no additional configuration. Commands are automatically registered when the plugin is loaded.
