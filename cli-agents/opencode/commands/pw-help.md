# /pw-help

Display help information about cali-product-workflow.

## Usage

```
/pw-help
```

## What it shows

- List of available commands
- Current workflow phase explanation
- Links to documentation

## Available Commands

| Command | Description |
|---------|-------------|
| `/pw-start` | Start a new workflow |
| `/pw-menu` | Show interactive workflow menu |
| `/pw-status` | Show current workflow status |
| `/pw-next` | Advance to next phase |
| `/pw-pause` | Pause the current workflow |
| `/pw-resume` | Resume a paused workflow |
| `/pw-stop` | Stop the current workflow |
| `/pw-ls` | List all workflows |
| `/pw-goto` | Switch to a different workflow |

## Workflow Phases

1. **Setup** - Gather requirements, clarify scope
2. **Context** - Understand the problem space
3. **Shape** - Shape the solution (Hill Chart, Rabbit Holes)
4. **Critique** - Adversarial review, identify gaps
5. **Gate** - Visual review via Plannotator
6. **Scope** - Break into typed scopes
7. **Interface** - Interface brainstorming (if needed)
8. **Int.Gate** - Interface review gate
9. **Selection** - Select approach
10. **Planning** - Create execution plan
11. **Execution** - Implement and validate
